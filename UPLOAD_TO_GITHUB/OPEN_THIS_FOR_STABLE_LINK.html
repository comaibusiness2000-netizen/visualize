Visualize: link stabile per iPhone

Questa guida serve per ottenere un link permanente apribile da telefono e aggiornabile automaticamente.

Parti da qui:

Apri START_HERE_STABLE_LINK.bat. Ti apre GitHub e poi ti guida nel collegare il repository.

Cartella pronta da caricare:

UPLOAD_TO_GITHUB

Carica il contenuto di questa cartella su GitHub. Non caricare node_modules.

Metodo ancora piu semplice se vuoi usare Git dal PC:
  1. Crea il repository su GitHub.
  2. Copia l'URL del repository, tipo https://github.com/TUO-UTENTE/visualize-app.git.
  3. Apri 3-publish-stable-link.bat.
  4. Incolla l'URL quando richiesto e completa il login GitHub.
  5. Per aggiornare il sito in futuro, apri update-live-site.bat.

Metodo consigliato: GitHub Pages

  1. Apri github.com/new.
  2. Crea un repository chiamato visualize-app.
  3. Nel repository, usa Add file -> Upload files.
  4. Trascina dentro tutti i file e cartelle contenuti in UPLOAD_TO_GITHUB.
  5. Premi Commit changes.
  6. Vai su Settings -> Pages.
  7. In Build and deployment, scegli GitHub Actions.
  8. Vai su Actions e attendi il deploy Deploy Visualize Preview.
Il link finale sara simile a:

https://TUO-NOME.github.io/visualize-app/

Aggiungerla alla schermata Home

  1. Apri il link finale in Safari su iPhone.
  2. Tocca Condividi.
  3. Tocca Aggiungi alla schermata Home.
  4. Conferma Aggiungi.

Aggiornamenti automatici

Quando modifichiamo l'app, basta caricare le nuove modifiche nello stesso repository GitHub. GitHub Pages aggiornera automaticamente lo stesso link.
