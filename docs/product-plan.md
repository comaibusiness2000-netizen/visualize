# Visualize Product Plan

## Positioning

Visualize is a personal visualization app for goals, identity change, guided reflection, and audio mindset practice.

The product should be framed as self-development, journaling, and motivational visualization. It should not claim to diagnose, treat, or guarantee mental health outcomes.

## Core experience

1. User defines a desired future.
2. User uploads photos or chooses an AI style.
3. App creates visual scenes for goal identity.
4. App creates an anti-vision: the cost of staying the same.
5. App creates a personalized spoken script.
6. User returns daily for review, listening, and small actions.

## MVP features

- Goal onboarding.
- Vision board.
- Photo upload placeholder.
- AI image prompt generator.
- Anti-vision reflection.
- Personalized script.
- Audio playback or text-to-speech.
- Progress check-ins.
- Premium paywall.

## Safety rules

- No graphic deathbed imagery.
- No humiliating or abusive language.
- No medical or psychological claims.
- Add intensity controls for anti-vision.
- Add opt-out and reset options.
- Use content moderation before sending prompts to image generation.

## Monetization

Free:

- 3 goals.
- Limited image generations.
- Basic scripts.

Premium:

- More image generations.
- Advanced future scenes.
- Anti-vision.
- Audio scripts.
- Cloud backup.
- Multiple routines.

Suggested launch price:

- Monthly: 9.99 EUR.
- Annual: 59.99 EUR.

For iOS, digital features consumed inside the app should use Apple In-App Purchase / subscriptions.
