# Things You Can Do Without Apple Developer

You can do these before paying for the Apple Developer Program.

## Decide product details

1. Final app name.
2. Backup app name if `Visualize` is unavailable.
3. Support email.
4. Public seller preference:
   - personal legal name, if enrolling as an individual;
   - company name, if enrolling as an organization.

## Review App Store copy

Open `docs/app-store-prep.md` and review:

1. Subtitle.
2. Description.
3. Keywords.
4. Category.
5. TestFlight "What to test".

## Test the web preview

Open:

`https://comaibusiness2000-netizen.github.io/visualize/?v=50`

Check:

1. First-run profile setup.
2. Life clock.
3. Daily tasks and long-term goals.
4. Vision image deck.
5. Anti-vision image deck.
6. Self Speech.
7. Language switching.
8. Dark mode contrast.

## Prepare tester list

Make a short list of beta testers:

1. First tester: `comaisamuele@gmail.com`.
2. Any friends who can test seriously.
3. Their iPhone/iPad model if they know it.

## Prepare screenshots later

When TestFlight is live, capture clean screenshots from the real app:

1. Onboarding.
2. Life clock.
3. Goals.
4. Vision.
5. Anti-vision.
6. Self Speech.
7. Profile/settings.
